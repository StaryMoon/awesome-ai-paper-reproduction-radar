# Contributing

Contributions are welcome. This radar is meant to help readers discover paper-to-code reproduction entries quickly.

Good contributions:

- add a missing high-interest AI paper reproduction entry;
- correct paper links, arXiv ids, tags, or categories;
- improve the searchable static page;
- add official-code links when authors release them;
- improve metadata refresh scripts.

Please keep every third-party implementation clearly marked as official or unofficial. Do not copy figures or tables from papers into this repository unless their license explicitly allows redistribution.
