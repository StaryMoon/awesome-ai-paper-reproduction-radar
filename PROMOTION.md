# Promotion Copy

Use these short launch drafts when sharing the project.

## Hacker News / Show HN

Title:

```text
Show HN: A searchable radar of AI paper reproduction repos
```

Body:

```text
I built a small searchable radar for high-interest AI papers and unofficial PyTorch reproduction repositories:

https://github.com/StaryMoon/awesome-ai-paper-reproduction-radar

It currently tracks 61 reproduction repos across frontier LLMs, reasoning RL, VLMs, video/world models, 3D/rendering, robotics, image restoration, segmentation, code models, and efficient sequence models.

The useful part is that it is not only a flat awesome list. It has:

- machine-readable JSON metadata;
- curated tracks such as Reasoning RL, VLM agents, Video/World Models, Robotics, and Restoration;
- a searchable static page;
- scheduled GitHub metadata refresh;
- paper/repo/status/tags in one table.

I made it because papers move much faster than official code releases, and I wanted one place to scan what is worth reading or reproducing next.
```
