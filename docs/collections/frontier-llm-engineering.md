# Frontier LLM Engineering Pack

For readers who care about MoE, long context, compact reasoning models, synthetic data, and attention-level engineering.

**Good for:** LLM infrastructure learners, model architecture readers, and people comparing technical reports.

**Why now:** Most technical report readers do not need a full training run first; they need a structured implementation map and citation trail.

## Suggested Reading Flow

1. Start from the first two repositories to understand the main architectural vocabulary.
2. Read the middle entries as design variants and ablations.
3. Use the later entries to compare adjacent tasks, data assumptions, or evaluation setups.
