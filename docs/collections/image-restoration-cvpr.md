# Image Restoration and Low-Level Vision Pack

For readers in deraining, dehazing, desnowing, low-light enhancement, restoration diffusion, and flow-based restoration.

**Good for:** Low-level vision researchers, restoration students, and people following CVPR restoration papers.

**Why now:** Restoration work is easier to buy into when papers are grouped by degradation, prior, and interface rather than by conference only.

1. Start from the first two repositories to understand the main architectural vocabulary.

3. Use the later entries to compare adjacent tasks, data assumptions, or evaluation setups.
