# VLM, Multimodal, and Agent Interface Pack

For readers following the path from visual understanding to tool-use, GUI agents, document understanding, and unified generation.

**Good for:** VLM students, multimodal agent builders, and researchers looking for cross-modal repo templates.

**Why now:** VLM work is fragmented across understanding, generation, action, and agent tasks; this track groups them by reusable interface shape.
