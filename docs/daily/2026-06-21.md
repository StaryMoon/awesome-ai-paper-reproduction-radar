## What Readers Should Take Away

The radar currently works best as a fast paper-to-starter map. It does not yet claim to be a complete official-code tracker. When a row says that the official link is not indexed, it means exactly that: the radar has not stored an official URL for the entry.

## Good Next Data Pass

The most useful next pass is a schema upgrade:

- `official_code_url`
- `official_code_status`
- `starter_type`

Once those fields exist, the README can show cleaner buckets such as `official indexed`, `official not indexed`, and `unofficial starter available`.
